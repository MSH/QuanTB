Перед использованием QuanTB выполните следующие шаги для настройки программы.

1. Перейдите в пункт меню "Препараты и режимы" и выберите "Изменить препараты".

  • QuanTB устанавливается с полным списком препаратов из каталога Global Drug Facility (GDF).
  • Для обновления списка препаратов, обратитесь к руководству пользователя QuanTB.

2. Перейдите в пункт меню "Препараты и режимы" и выберите "Изменить режимы лечения".

  • QuanTB содержит примеры режимов лечения только для учебных целей.
  • В руководстве пользователя QuanTB указано как внести необходимые Вам режимы лечения.

3. После обновления препаратов и режимов лечения, QuanTB готова к использованию.

  • Для создания нового расчёта(прогноза), перейдите в меню "Файл" и выберите "Новый".
  • Для открытия уже существующего расчёта(прогноза), перейдите в меню "Файл" и выберите "Открыть...".
При работе с QuanTB не забывайте сохранять данные прогнозов. QuanTB не сохраняет эти данные автоматически.

Перейти к руководству пользователя QuanTB...
