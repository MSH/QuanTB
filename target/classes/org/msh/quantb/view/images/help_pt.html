Antes de começar a usar o QuanTB, siga os passos abaixo para adaptá-lo ao seu local ou programa.

1. Vá para o menu Medicamentos & Regimes e selecione Editar medicamentos.

  • QuanTB é instalado com todos os medicamentos do catálogo do Global Drug Facility (GDF).
  • Se esta lista de medicamentos não está compatível com o seu contexto local, você deve atualizá-la. Consulte o Guia do Usuário do QuanTB para obter ajuda.

2. Vá para o menu Medicamentos & Regimes e selecione Editar regimes de tratamento.

  • QuanTB contém exemplos de regimes de tratamento para efeito de treinamento apenas.
  • Consulte o Guia do Usuário do QuanTB para obter informações sobre como inserir seus regimes de tratamento.

3. Após atualizar as Medicamentos e Regimes de Tratamento, você está pronto para começar a usar o QuanTB.

  • Para criar uma nova quantificação (previsão), vá para o menu Arquivo e selecione Novo.
  • Para abrir uma quantificação (previsão) salva, vá para o menu Arquivo e selecione Abrir.
Conforme você usar o QuanTB, salve seu trabalho regularmente. QuanTB não salva os dados automaticamente.
